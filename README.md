NASA Asteroid Impact Simulator for NASA SpaceApp Challenge 2025

Allows users to simulate impact consequences of **NEO (Near-Earth Objects)** and select proper countermeasures

[**Purrfect Impact**](https://levilleza.github.io/purrfect-impact/)
